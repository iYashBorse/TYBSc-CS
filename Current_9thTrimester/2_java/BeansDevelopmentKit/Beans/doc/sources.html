BDK sources

The BDK includes a variety of different source code:
  1. The "beanbox\sun\beanbox" sub-directory include the complete sources for the BeanBox container. This example code may be freely recycled and reused "at your own risk". See the BDK Licensing terms for details.
  2. The "demo\sunw\demo\*" sub-directories include the complete sources for the various example beans and sample services.  This example code may be freely recycled and reused "at your own risk". See the BDK Licensing terms for details.
