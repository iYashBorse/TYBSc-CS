Wrapped beans

Simple HTML page to show three beans running inside the generic BeanWrapper applet.

The trhee beans are a juggler bean, a molecule viewer bean, and a serialized sorter bean. The juggler and the sorter are both beans and applets, whereas the molecule view is simply a bean.

Try running this page with the JDK 1.1 appletviewer, or with HotJava 1.0.
